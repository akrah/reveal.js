Installation
============

```bash
$ wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
$ source ~/.bashrc
$ nvm install node
$ sudo apt install pandoc
$ npm install --global gulp-cli
```

Dans reveal.js/ :

```bash
$ npm update
$ git submodule init
$ git submodule update
```
